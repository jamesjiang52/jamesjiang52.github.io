Bitwise

Source code available here. Documentation available here.

Bitwise was my first ever Python project that I intended to release and be used by other people, and so is technically my first true open source project.

Bitwise is a Python library intended to make hardware design and simulation more accessible for software engineers. While it can never replace a true hardware description language, it aims to serve as a useful tool in the hardware design process, allowing a user to build and test their digital circuits in a high-level programming language (i.e. with simpler syntax and more flexible semantics) before implementing them using an HDL.

I started Bitwise in the Fall of 2018, when I was in second year at the University of Toronto. One of the courses I was taking was Digital Systems, which was an introductory course on computer hardware - logic gates, registers, combinational and sequential circuits, and so on. Since I had exposure to such things already, I decided to challenge myself by starting this project. Initially, my goal wasn't to create a Python library; instead, I was inspired by the megaprocessor to create a microprocessor from scratch. Along the way, I decided that releasing a Python package to encourage others to do the same would likely be a better path to follow.

Example

The following code creates a half-adder.

"""
Create a half-adder.
"""

import bitwise as bw

def main():
    # initialize inputs
    a = bw.wire.Wire()
    b = bw.wire.Wire()

    # initialize outputs
    sum_ = bw.wire.Wire()
    carry_out = bw.wire.Wire()

    # create circuit
    bw.gate.XORGate2(a, b, sum_)  # XORs a and b and puts the result into sum_
    bw.gate.ANDGate2(a, b, carry_out)  # ANDs a and b and puts the result into carry_out

if __name__ == "__main__":
    main()
